### 발견된 이슈 (BUG-001)

**공개 프로필 직접 URL 접근 시 "사용자를 찾을 수 없습니다" 표시**
- **원인**: 시드 유저 데이터가 서버 사이드에서 사용 불가 (localStorage 기반 시뮬레이션의 한계)
- **재현**: /users/user-alpha-2 등 시드 유저 URL 직접 접근 시 404 표시
- **영향**: 유저 목록 페이지(클라이언트 렌더링)에서는 정상 표시되나, 직접 URL 접근 시 실패
- **심각도**: Medium — 데모 흐름에서 유저 목록 → 프로필 클릭 순서로 접근하면 정상 동작
- **권장 수정**: SSR 시점에서도 시드 데이터를 가용하도록 `getSeededUsers()` 함수 수정
